Welcome to the Ultimate Soccer Team DB

About

This Flask application is supposed to offer you an overview of fictional soccer teams.
This means, there will be parts you know from the real world like player names, on the other hand the only title you can win in our fictional database, is the UEFA Champions League.
There may also be player with names or teams with number of titles, you may never have heard of. ;)
The idea behind this database is to give information concerning Teams at a certain point of time.
Users will be able to see stats by querying predefined SQL-Queries. Besides they may perform CRUD operations to edit the data.
The initial load of the database will fill the database with sample data.

Congratulations! By setting the application up, you have passed the hardest part. Now have fun exploring our database and queries. Enjoy!

PS: in Github you will find a cool walkthrough video of the database, functions and queries :)

Error Handling

Check the User Guide in Github.

Make sure you use the correct required python packages.

Make sure you have your local pgadmin passwords for your pgadmin server/user inserted.

Make sure you created a database in pgadmin called "soccer".

In some cases it makes sense to delete all tables from the database "soccer". (Only in case they exist already)

Deleting pycache makes sense sometimes as well.

The application has been tested on two Windows 10 computers.
